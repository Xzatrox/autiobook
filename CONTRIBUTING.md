# contributing

## development workflow

1. add task to ROADMAP.md before starting
2. write tests before implementation
3. run `make precommit` before committing
4. update DESIGN.md after architectural changes
5. update README.md after user-visible changes
6. mark task complete in ROADMAP.md

## build

```bash
make
```

## test

```bash
make test
```

## precommit

```bash
make precommit
```
